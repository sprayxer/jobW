Региональность

Москва/МО
  • проработали семантическое ядро на заранее оговоренные категории товаров;
  • подключили РСЯ с адаптивными креативными баннерами и видеообъявлениями;
  • разработали кампанию ретаргетинга по необходимым условиям возврата целевых пользователей на сайт;
  • исходя из аналитических данных внедрили корректировки показов объявлений для аудиторий (полу, возрасту), по времени заказов, по устройствам (мобильные, десктопные);
  • выполнили тестирование различных стратегий показов для поиска наиболее выгодного решения;
  • исключили неконверсионные площадки исходя из аналитических данных в ходе ежемесячной оптимизации;
  • исключили нецелевые поисковые запросы на основе аналитических данных по всем запросам за прошлый период работы рекламы (этим самым основной трафик изменился на целевой);
  • еженедельно оптимизировали работу кампаний для исключения нецелевых показов объявлений;
  • настроили магазин на Yandex.Market по моделям FBS, DBS;
  • настроили магазин на маркетплейсе СбермегаМаркет;
  • дополнительно запустили рекламную кампанию в Yandex.Direct "Смарт-баннеры" и создали адаптивные креативные баннеры с динамическим контентом для показов в сетях и товарной галерее.
